# Gin-X
> Golang WEB framework base on [Gin](https://github.com/gin-gonic/gin), more components to improve development efficiency.


## Components

- Slim router

- Cron

- Config

- Database

- Enum

- Helper

- Log

- Middleware

- ORM Model

- Pager

- Redis

- Response
